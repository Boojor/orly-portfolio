# Spacing & Layout

The project utilizes Finsweet Client-First spacing rules, preserved inside Tailwind 4 `@theme` configuration.

## Base Spacing Scale

These translate natively to Tailwind `p-*`, `m-*`, `gap-*`, etc. values (e.g. `--spacing-large` -> `p-large`).

## Section Spacing

Sections use designated padding wrappers (`.padding-section-large`, `.padding-section-medium`, etc.) to control vertical rhythm responsively across breakpoints.
These values dynamically shrink down for mobile viewports.
