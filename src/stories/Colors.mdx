# Color Architecture

The portfolio's color system follows a strict logical division:
1. **Global & Persistent Colors**: Core neutral values used sitewide for backgrounds, text, and structural UI elements.
2. **Page-Specific Accents**: Thematic color palettes that change depending on the page (e.g., the Homepage has its distinct accents, and each Case Study gets its own branded identity).

<br/>
